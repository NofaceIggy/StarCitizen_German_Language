Anleitung zur Installation der Datei:

Erstelle alle nötigen neuen Ordner, damit aus dem RSI-Ordner heraus folgenden Verzeichnisbaum vorhanden ist: 

\StarCitizen\LIVE\data\Localization\german_(germany)

Legen Sie Ihre zuvor übersetzte global.ini-Datei in den neuen Ordner.

Fügen Sie die nachfolgend aufgeführte Zeile in Ihre „user.cfg“-Datei in Ihrem \StarCitizen\LIVE Ordner ein:

g_language = german_(germany)

Starte das Spiel wie gewohnt und die Lokalisierung sollte funktionieren.
